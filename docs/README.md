RedGNAT Docs
============

Documentation is organised using the [Diátaxis](https://diataxis.fr/) model. Each document has one primary purpose — choose the section that matches your intent.

Structure
---------

| Section | Purpose | Contents |
|---------|---------|----------|
| [tutorials/](tutorials/README.md) | Learning by doing | End-to-end walkthroughs |
| [how-to/](how-to/README.md) | Solving a task | Task-oriented operating recipes |
| [reference/](reference/README.md) | Exact technical details | Configuration syntax, API endpoints, technique library |
| [explanation/](explanation/architecture/architecture.md) | Design rationale | Architecture, safety model, phase model, feedback loop |
